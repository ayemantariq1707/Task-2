# Random Flashcard Quiz App

An interactive application that helps users learn using randomly generated flashcards and quiz-based learning. 

---
## Features

-  Random flashcard generation
-  Flip cards to reveal answers
-  Quiz mode for self-assessment
-  Instant feedback on answers
-  Simple and user-friendly interface
-  Fast and responsive performance
---
## 🛠️ Tech Stack
- Android Studio 
- Kotlin 
- XML 
- Local storage / API integration
---
## How It Works

- The app displays a random question as a flashcard
- Users can flip the card to view the answer
- In quiz mode, users attempt to answer before revealing

- Practice randomization logic
- Improve UI/UX design skills
